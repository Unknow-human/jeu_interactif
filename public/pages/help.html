Numéno 4

Aide

Le but du jeu est de deviner un code secret de 4 chiffres.

Après chaque tentative, vous recevrez des indications :

  • Chiffres corrects et bien placés : Le nombre de chiffres que vous avez bien positionnés.
  • Chiffres corrects mais mal placés : Les chiffres corrects mais à la mauvaise position.

Le premier joueur à trouver le code exact remporte la partie !
