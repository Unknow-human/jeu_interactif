Numéno 4

Aide

Numéno 4 est un jeu passionnant qui vous défie de deviner un code secret de 4 chiffres. Voici un guide détaillé pour comprendre comment jouer, que ce soit en mode solo ou duel.

Mode Solo

En mode solo, votre objectif est de deviner le code secret généré par l'ordinateur. Voici comment jouer :

  • Entrez un code à 4 chiffres dans la zone de saisie située en bas de l'écran.
  • Appuyez sur le bouton "Valider" pour soumettre votre proposition.
  • Après chaque tentative, l'historique des tentatives affichera les indices suivants :
    • Chiffres corrects et bien placés : Le nombre de chiffres que vous avez bien positionnés.
    • Chiffres corrects mais mal placés : Les chiffres corrects mais à la mauvaise position.
  • Continuez à faire des tentatives jusqu'à ce que vous découvriez le code secret.
  • Vous pouvez abandonner le jeu à tout moment en cliquant sur le bouton "Abandonner", ce qui vous ramènera à la page de sélection du mode de jeu.

Mode Duel

En mode duel, vous affrontez un autre joueur pour voir qui devinera le code secret en premier. Voici comment jouer :

  • Attendez que le jeu trouve un adversaire pour vous.
  • Entrez un code à 4 chiffres dans la zone de saisie située en bas de l'écran.
  • Appuyez sur le bouton "Valider" pour soumettre votre proposition.
  • Après chaque tentative, l'historique des tentatives affichera les indices suivants :
    • Chiffres corrects et bien placés : Le nombre de chiffres que vous avez bien positionnés.
    • Chiffres corrects mais mal placés : Les chiffres corrects mais à la mauvaise position.
  • Vous pouvez utiliser la zone de chat pour communiquer avec votre adversaire.
  • Le premier joueur à découvrir le code secret gagne la partie.
  • Vous pouvez abandonner le jeu à tout moment en cliquant sur le bouton "Abandonner", ce qui vous ramènera à la page de sélection du mode de jeu.

Changer de Thème

Vous pouvez changer le thème de l'application en cliquant sur le bouton "Changer de Thème". Les thèmes disponibles incluent :

  • Jeu Intergalactique
  • Étoile de Mer
  • Clair de Lune
  • Ciel Nuageux
  • Orageux
